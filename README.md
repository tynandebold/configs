# configs

prefs for how i like stuff

## misc.

- list vscode extensions:

  ```sh
  $ code --list-extensions
  ```

- location of iTerm profiles file:

  ```sh
  ~/Library/ApplicationSupport/iTerm2/DynamicProfiles
  ```

- location of z.sh and it's data file:

  ```sh
  ~/Bin/z.sh
  ~/.z
  ```
